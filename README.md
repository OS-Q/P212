### [设备描述](https://github.com/OS-Q/D145/wiki) 

LoRaWAN基于LoRa远距离通信网络设计的一套通讯协议和系统架构,按协议分层来说LoRaWAN就是MAC层,LoRa是物理层。LoRaWAN也被叫做LoRaMAC。
